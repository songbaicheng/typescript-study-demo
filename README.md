# 学习 TS 过程中的 demo
## 执行
安装 ts-node，使用 ```ts-node [file.ts]``` 查看ts文件执行结果。 
```node
npm i ts-node -g

npm i @types/node -D
```
## 练习目录
1. base-type.ts：基础类型
2. interface.ts：接口
3. function.ts：函数
4. type.ts：数据类型
5. class.ts：类
6. enum.ts：枚举
7. symbol.ts：唯一值
8. iterator.ts：迭代器
9. genericity.ts：泛型
10. mixins.ts：混入
